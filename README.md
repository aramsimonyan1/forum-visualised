BSc Data Science and Computing project

## To run the app:
###
    $go run main.go

    To run with docker:
    $docker image build -t my-forum-app .  
    $docker container run -p 8080:8080 my-forum-app

    Open your web browser and navigate to http://localhost:8080.
    Register user > Login > ...
